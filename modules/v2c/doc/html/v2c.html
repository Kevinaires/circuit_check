V2C: VHDL to C translator

 v2c is a tool that automatically translates a VHDL source into an equivalent C program, i.e. a C function with the same input-output behaviour of the digital circuit the VHDL represents.

It has been realized by two students as an exam task, and is far from being a state-of-the-art product...

v2c can cope with data-flow and behavioural architectures, sequential or combinatorial, with most of the sequential and concurrent statements supported.

The goals of the project are the following:

  • Fidelity: the C code behaves exactly like the VHDL source.
  • Readability: the code, although fairly complex, is human-readable and modifiable.
  • Observability: the circuit internal state (if one exists) can be easily observed.
  • Controllability: the internal state can be easily modified.
  • Flexibility: several switches lets you alter the translation to suit hardware and software needs.
The tool is easy to use and comes with an interface, called v2cx, that can analyse the input and give the appropriate arguments to v2c.

More information is available on-line: some examples, the translation grammar and the input format.

A postscript italian manual is also available.
